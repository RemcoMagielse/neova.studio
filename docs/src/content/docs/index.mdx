## Features

<CardGrid>
  <Card title="Task Management" icon="list-format">
    <img src="/assets/img/neova-stages.png" alt="Task Management" class="feature-image" />
    Kanban-style boards to track development tasks through customizable stages. Organize your workflow with drag-and-drop simplicity.
  </Card>
  <Card title="Prompt Builder" icon="pencil">
    <img src="/assets/img/neova-taskeditor-full.png" alt="Prompt Builder" class="feature-image" />
    Create structured, context-aware prompts for AI coding agents. Build comprehensive instructions with templates and dynamic replacements.
  </Card>
  <Card title="Snippet Library" icon="document">
    <img src="/assets/img/neova-snippets.png" alt="Snippet Library" class="feature-image" />
    Store and reuse code snippets, templates, and instructions. Quick access to your most-used patterns and boilerplate.
  </Card>
  <Card title="Template System" icon="setting">
    <img src="/assets/img/neova-templates.png" alt="Template System" class="feature-image" />
    Build reusable templates with dynamic replacements. Variables, conditionals, and smart context injection for any workflow.
  </Card>
</CardGrid>
