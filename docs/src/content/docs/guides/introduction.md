Neova is a desktop application designed to supercharge your AI-assisted development workflow. It bridges the gap between human developers and AI coding assistants through context-aware instruction generation.

## What is Neova?

Neova helps you:

- **Organize tasks** with kanban-style boards and customizable stages
- **Build prompts** with structured, context-aware templates for AI agents
- **Manage snippets** for reusable code patterns and instructions
- **Create templates** with dynamic variable replacements

## Who is Neova for?

Neova is built for developers who work with AI coding assistants like:

- Claude Code
- GitHub Copilot
- Cursor
- Other AI-powered development tools

If you find yourself repeatedly crafting prompts, managing context, or organizing your AI-assisted workflow, Neova is for you.

## Key Features

### Task Management
Track your development tasks through customizable stages. Move tasks through your workflow with drag-and-drop simplicity.

### Prompt Builder
Create comprehensive prompts with templates, snippets, and dynamic replacements. Build context-aware instructions that help AI assistants understand your codebase.

### Snippet Library
Store frequently used code patterns, instructions, and boilerplate. Quick access to your most-used content with search and tags.

### Template System
Build reusable templates with variables and conditionals. Smart context injection for any workflow.
