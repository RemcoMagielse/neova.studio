# Neova

AI-powered development companion for coding agents.

**Think. Build. Beyond.**

## About

Neova is a desktop application that helps developers organize AI-assisted development workflows. It bridges the gap between human developers and AI coding assistants through context-aware instruction generation.

### Features

- **Task Management** — Kanban-style boards with customizable stages
- **Prompt Builder** — Structured, context-aware prompts for AI agents
- **Snippet Library** — Reusable code patterns and instructions
- **Template System** — Dynamic variable replacements and smart context injection

## Download

Download the latest release for macOS from [GitHub Releases](https://github.com/RemcoMagielse/neova.studio/releases).

Requires macOS 12.0 or later.

## Documentation

Visit [neova.studio](https://neova.studio) for full documentation.

## Development

This repository contains the public website and release downloads. The docs site is built with [Astro](https://astro.build) and [Starlight](https://starlight.astro.build).

```bash
cd docs
npm install
npm run dev
```
